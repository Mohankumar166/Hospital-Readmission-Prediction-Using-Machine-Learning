# 🏥 Hospital Readmission Prediction Using Machine Learning

A machine learning project that predicts whether a patient is likely to be readmitted to the hospital within 30 days. The system analyzes demographic and clinical data, applies data preprocessing and feature engineering, and uses a *Random Forest Classifier* to generate accurate predictions.

## 🚀 Features

* Predicts 30-day hospital readmission risk
* Data preprocessing and feature engineering
* Random Forest-based classification model
* Performance evaluation using Accuracy, Precision, Recall, and F1-Score
* Clean and modular Python implementation

## 🛠️ Technologies Used

* Python
* Pandas
* Scikit-learn
* VS Code
* Git & GitHub

## 📊 Dataset

The model is trained on hospital patient records containing features such as:

* Age
* Gender
* Blood Pressure
* Cholesterol
* BMI
* Diabetes Status
* Hypertension Status
* Medication Count
* Length of Hospital Stay
* Discharge Destination

## 🎯 Objective

To help healthcare providers identify patients at high risk of readmission, enabling early intervention and improving patient outcomes while reducing healthcare costs.
