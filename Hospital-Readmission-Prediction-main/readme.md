

 Hospital Readmission Prediction Using Machine Learning

📌 Project Description

Hospital readmissions within 30 days are a major concern for healthcare systems as they increase costs and indicate possible gaps in patient care. This project uses **machine learning techniques** to predict whether a patient is likely to be readmitted within 30 days based on demographic and clinical data.



🎯 Problem Statement

Manual analysis of patient medical records to identify readmission risk is complex and time-consuming. An automated prediction system can help hospitals take preventive measures and improve patient outcomes.



 🎯 Objectives

* To analyze patient health and hospital stay data
* To preprocess and encode medical records
* To build a machine learning model for readmission prediction
* To evaluate model performance using standard metrics



 📊 Dataset Description

The dataset consists of hospital patient records containing demographic information, medical conditions, and hospital stay details.

Features include

* Age
* Gender
* Blood Pressure
* Cholesterol
* BMI
* Diabetes
* Hypertension
* Medication Count
* Length of Stay
* Discharge Destination

Target Variable:

`readmitted_30_days` (Yes / No)



⚙️ Methodology

1. Data loading and cleaning
2. Feature engineering (splitting blood pressure)
3. Encoding categorical variables
4. Model training using Random Forest
5. Performance evaluation



 🛠 Tools & Technologies Used

* Python
* Pandas
* Scikit-learn
* VS Code
* GitHub



🤖 Machine Learning Model Used

 Random Forest Classifier



 📈 Performance Evaluation

Evaluation Metrics:

* Accuracy
* Precision
* Recall
* F1-Score

The model achieved good accuracy, demonstrating the effectiveness of machine learning in predicting hospital readmissions.
